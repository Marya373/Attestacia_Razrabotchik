1. Создание двух новых директорий:
```
mkdir "Игрушки для школьников"
mkdir "Игрушки для дошколят"
```

2. Создание текстовых файлов в директории "Игрушки для школьников":
```
cd "Игрушки для школьников"
touch "Роботы" "Конструктор" "Настольные игры"
```

3. Создание текстовых файлов в директории "Игрушки для дошколят":
```
cd ..
cd "Игрушки для дошколят"
touch "Мягкие игрушки" "Куклы" "Машинки"
```

4. Объединение двух директорий в одну "Имя Игрушки":
```
cd ..
mkdir "Имя Игрушки"
mv "Игрушки для школьников" "Имя Игрушки/"
mv "Игрушки для дошколят" "Имя Игрушки/"
```

5. Переименование директории "Имя Игрушки" в "Игрушки":
```
mv "Имя Игрушки" "Игрушки"
```

6. Просмотр содержимого каталога "Игрушки":
```
cd "Игрушки"
ls
```

7. Установка и удаление snap-пакета:
```
sudo snap install <название_пакета>
sudo snap remove <название_пакета>
```

8. Добавление задачи для выполнения каждые 3 минуты записи в текстовый файл или копирования из каталога А в каталог Б:
```
crontab -e
```
Откроется файл конфигурации Cron. Необходимо добавить строку в следующем формате:
```
*/3 * * * * <команда>
```
Например, для записи текста в файл:
```
*/3 * * * * echo "Текст записываемый в файл" >> /путь/к/файлу.txt
```
Или для копирования из каталога А в каталог Б:
```
*/3 * * * * cp -r /путь/к/каталогуА/* /путь/к/каталогуБ/
